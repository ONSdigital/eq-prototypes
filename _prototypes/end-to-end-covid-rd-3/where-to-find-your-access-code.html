Where to find your access code

To start your census online, you will need your access code printed on the invitation letter we sent you. Your 16-character access code will be a combination of letters and numbers.

!Keep this code safe. You will need to enter it every time you access your census.

Access codes requested for you

Someone you live with can request an access code for you, which can be sent by text message or post.

The text will be sent from "Census 2021", and includes your 16-character access code. The letter with your code will be addressed to "Individual Resident".
