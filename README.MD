# eQ prototypes


## Install
- `gem install bundler`
- `git clone https://github.com/ONSdigital/eq-prototypes.git`
- `cd eq-prototypes`
- `bundle install`

## Run

- `bundle exec jekyll serve`
- run `gulp` in a separate tab
